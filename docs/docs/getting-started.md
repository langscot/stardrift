# Getting Started

## 1. Join a Stardrift Server

You need to be in a Discord server that has the Stardrift bot installed. The bot's home server — **Sol Nexus** — is where all new players start. If you're reading this, you're probably already there.

## 2. Your First Interaction

The moment you use any Stardrift command, your pilot account is created automatically. You'll be placed at **Sol Nexus** (0, 0) with:

| Stat | Starting Value |
|------|---------------|
| Credits | 0 |
| Fuel | 100 / 100 |
| Cargo Capacity | 1,000 units |

No setup required — just run a command.

## 3. Open the Menu

```
/menu
```

The main menu gives you an overview of your current status and quick-access buttons to every game section: cargo, map, mining, selling, and travel.

## 4. Mine Some Ore

Find a planet or asteroid belt channel in the current server (e.g. `#planet-1-ignis` or `#asteroid-belt`), then run:

```
/mine
```

Your ship drills for ore and deposits it into your cargo hold. Mining has a **30-second cooldown** — use the **Mine Again** button that appears after a successful mine to queue up your next drill quickly.

Your cargo hold has a capacity limit. Keep an eye on it with `/inventory`.

## 5. Sell Your Haul

Head to the `#station-market` channel and run:

```
/sell
```

You'll see current NPC prices for each ore type in your cargo. Hit **Sell All** to dump everything at once, or select individual items to sell specific quantities.

Prices fluctuate based on supply — the more a resource gets sold into a market, the lower the price drops. Prices recover over time.

## 6. Travel to Another System

Go to `#travel-hub` and run:

```
/travel
```

You'll see nearby systems sorted by distance (in light-years), the fuel required to reach each one, and the estimated travel time. Select a destination and confirm departure.

While in transit you're locked out of game actions but can still chat. You'll get a notification when you arrive.

---

## 7. Upgrade Your Ship

Once you've earned some credits, run:

```
/shop
```

Browse ships and modules. Start with a **Mk1 Laser** (2,000¢) to boost your mining yield, or save up for a bigger ship with more module slots. Use `/loadout` to fit purchased modules into your ship's slots.

---

## The Core Loop

```
Mine ore  →  Sell at market  →  Upgrade ship & modules  →  Travel to richer systems  →  Repeat
```

As you accumulate credits, upgrade your lasers for better yield, fit scanners to discover rare anomalies, expand your cargo hold, and invest in faster ships to reach distant systems.
