# Commands Reference

All Stardrift interactions use Discord slash commands. Type `/` in any channel to see available commands.

---

## Gameplay Commands

<SlashCommand name="mine" description="Mine ore from your current location.">

- Use in a **planet channel** or **asteroid belt channel**
- 30-second cooldown between mines
- Adds ore directly to your cargo hold
- Blocked if your cargo is full or you're in transit

</SlashCommand>

<SlashCommand name="inventory" description="View your current cargo and station storage.">

- Shows cargo hold (used / capacity)
- Shows station storage at your current system
- Shows current fuel level

</SlashCommand>

<SlashCommand name="sell" description="Open the NPC market and sell cargo.">

- Use in `#station-market`
- Shows current prices per ore type
- **Sell All** button dumps your entire cargo
- Individual item buttons let you sell specific stacks
- Prices shown reflect any proxy tax if applicable

</SlashCommand>

<SlashCommand name="travel" description="View nearby systems and initiate a jump.">

- Use in `#travel-hub`
- Lists nearby enrolled systems sorted by distance
- Shows fuel cost and travel time for each destination
- **Confirm** button starts the jump
- You're locked out of game actions while in transit

</SlashCommand>

<SlashCommand name="map" description="Display a star map of systems near your current location.">

- Shows up to 15 systems within 50 light-years
- Lists coordinates, star type, resource rating, and distance
- Useful for planning travel routes

</SlashCommand>

<SlashCommand name="menu" description="Open the main navigation menu.">

- Shows your current status (location, fuel, credits)
- Buttons to quickly access cargo, map, mining, selling, and travel sections
- Available from any channel

</SlashCommand>

<SlashCommand name="shop" description="Browse and buy ships and modules.">

- Opens the module shop by default
- **Ships tab** — browse ship hulls, compare slot counts and base stats
- **Modules tab** — browse lasers, scanners, cargo expanders, and utility modules
- Filter modules by category
- Use the dropdown to select and purchase items

</SlashCommand>

<SlashCommand name="loadout" description="View and manage your ship's module slots.">

- Shows your current ship and all fitted modules
- **Combined stats** summary at the top
- **Fit**, **swap**, or **unfit** modules in each slot
- Shows your unfitted module inventory

</SlashCommand>

<SlashCommand name="prospect" description="Survey the resources of your current star system.">

- Only visible to you (ephemeral reply)
- **In a planet channel** — shows that planet's ore yields with a select menu to navigate to other bodies
- **In an asteroid belt channel** — shows that belt's richness and ore yields with navigation
- **Anywhere else** — shows a full system report (all planets + belts)
- Use the **← System Overview** button or the dropdown to jump between bodies

</SlashCommand>

## Admin Commands

These commands require **Administrator** permission in the Discord server.

<SlashCommand name="enroll" description="Claim an unenrolled system for your server." admin>

- Registers the system in the galaxy database
- Creates all gameplay channels automatically
- Links the server's guild ID to the system's galactic coordinates

</SlashCommand>

<SlashCommand name="setup-hub" description="Register Sol Nexus (the central hub) as a proxy in your server." admin>

- Gives your server access to Sol Nexus channels
- Proxy penalties apply (70% mining yield, 80% sell rate)
- Useful for servers that want a gateway to the hub without full enrollment

</SlashCommand>

<SlashCommand name="add-proxy" description="Mirror another enrolled system into your server." admin>

- Creates proxy channels for a system your server doesn't own
- Players access the system with proxy penalties
- Useful for multi-system server setups

</SlashCommand>

<SlashCommand name="systems" description="List all systems registered on the current Discord server." admin>

- Shows home systems and proxy systems
- Includes resource ratings and enrollment dates

</SlashCommand>

## Button Actions

Many commands produce interactive buttons. The common ones:

| Button | Context | Action |
|--------|---------|--------|
| **Mine Again** | After mining | Re-runs <Cmd>/mine</Cmd> immediately |
| **Sell All** | <Cmd>/sell</Cmd> | Sells your entire cargo hold |
| **Sell [Item]** | <Cmd>/sell</Cmd> | Sells a specific item stack |
| **Confirm Departure** | <Cmd>/travel</Cmd> | Starts a jump to the selected system |
| Menu buttons | <Cmd>/menu</Cmd> | Navigate to game sections |
| **← System Overview** | <Cmd>/prospect</Cmd> | Back to the body select menu |
| Body select menu | <Cmd>/prospect</Cmd> | Switch between planets, belts, or full report |
